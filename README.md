# Quant Finance Portfolio

A portfolio of quantitative finance projects built while learning derivatives pricing, options theory, and financial modeling. Each project is self-contained with its own code, README, and dependencies.

---

## Projects

### [Project 1 — Derivatives Payoff Visualizer](./project1_payoff_visualizer/)

Visualizes profit/loss at expiration for futures contracts, call options, and put options. Introduces the three foundational derivative instruments and the payoff diagram — the visual language every derivatives trader uses.

**Concepts:** Futures, call options, put options, payoff diagrams, risk/reward profiles
**Stack:** Python, NumPy, Matplotlib

---

### [Project 2 — Black-Scholes Options Pricer](./project2_black_scholes/)

Command-line pricer for European options using the Black-Scholes model. Prices calls and puts, computes the four main Greeks (delta, gamma, vega, theta), and verifies results with a put-call parity check.

**Concepts:** Black-Scholes model, geometric Brownian motion, risk-neutral pricing, the Greeks, put-call parity
**Stack:** Python, SciPy
